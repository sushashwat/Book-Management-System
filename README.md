
# Book-Management-System

A simple React-based Book Management System that allows users to search and browse books.

## 🚀 Features

* Search books by title
* Responsive book card layout
* Dynamic filtering using React state
* Clean and modern UI
* Component-based architecture

## 🛠️ Technologies Used

* React
* JavaScript
* CSS
* Vite

## 📂 Project Structure

```bash
src/
 ├── assets/
 │    └── utils/
 │         └── Mockdata.js
 │
 ├── components/
 │    ├── Book.jsx
 │    ├── BookList.jsx
 │    └── style.css
 │
 ├── App.jsx
 └── main.jsx
```
